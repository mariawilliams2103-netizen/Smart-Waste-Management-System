# Smart Waste Management System (SWMS)

## Overview

The **Smart Waste Management System (SWMS)** is a frontend prototype designed to demonstrate how technology can improve waste collection, monitoring, and management processes. The system provides a user-friendly interface for accessing waste management information and visualizing key system components.

This project serves as a static web application prototype suitable for academic projects, demonstrations, and future development into a full-scale smart waste management platform.

---

## Features

### Splash Screen

* Attractive welcome screen displayed on application launch.
* Automatically redirects users to the login interface.

### User Authentication

* Login modal with email and password fields.
* Password visibility toggle.
* Basic client-side validation.
* Demo access using any valid email and a password containing at least six (6) characters.

### Dashboard Interface

After successful login, users can access a dashboard containing:

* Home Section
* About the System
* Key Features
* Use Case Demonstrations
* System Architecture Diagrams
* Benefits of Smart Waste Management
* Contact Information

### Responsive Design

* Modern user interface.
* Mobile-friendly layout.
* Clean navigation experience.

## Getting Started

### Option 1: Open Directly

Simply open the `index.html` file in any modern web browser.

### Option 2: Run a Local Server

Using Python:

```bash
cd "Smart Waste"
python -m http.server 8000
```

Open your browser and visit:

```text
http://localhost:8000
```

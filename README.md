# Smart Waste Management System (SWMS)

A simple static frontend prototype for a Smart Waste Management System (SWMS). Includes a splash screen, login modal, and a demo dashboard layout.

## Features

- Splash screen with auto-launch to login
- Login modal with password toggle and basic validation
- Dashboard-style hero and information sections (About, Features, Use Case, Diagrams, Benefits, Contact)

## Quick start

1. Open the project folder and open `index.html` in your browser.
2. Or serve locally with Python:

```bash
cd "Smart Waste"
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Sign in using any email and a password of at least 6 characters to reveal the demo dashboard.

## Development notes

- Styles are in `style.CSS`.
- The splash → login → dashboard flow is implemented in `index.html` with simple client-side JS.
